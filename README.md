# Vellum — An Obsidian Theme

> *Write like you're inscribing on ancient parchment. Read like you're holding a manuscript under moonlight.*

A sophisticated dual-mode theme for Obsidian — warm **Vanilla Parchment** for daylight writing, calm **Charcoal Grey** for late-night sessions.

---

## Features

- **Dual mode** — Full light & dark theme support
- **Playfair Display** headings — elegant, editorial, timeless
- **Charm** body text — calligraphic, warm, a joy to read
- **Sepia toggle** — Switch light theme font from Navy Blue → Warm Brown (old scroll feel)
- **Ultra minimal UI** — Everything hidden until you need it
- **Style Settings support** — Adjust font size, line height, content width, accent colors, and more — no CSS knowledge needed
- **WCAG AA accessible** — All color combos tested (6.48:1 contrast ratio on dark mode)
- **Mobile responsive** — Works on Obsidian Mobile too

## Installation

### From Obsidian Community Themes (recommended)
1. Open **Settings → Appearance → Themes**
2. Search for **Vellum**
3. Click **Install and use**

### Manual Install
1. Download `theme.css` and `manifest.json`
2. Create a folder: `<vault>/.obsidian/themes/Vellum/`
3. Place both files inside
4. Go to **Settings → Appearance** and select **Vellum**

---

## Optional: Style Settings Plugin

Install the [Style Settings plugin](https://github.com/mgmeyers/obsidian-style-settings) to get a visual control panel for:

- 📏 Font size slider (13–22px)
- 📐 Line height slider (1.3–2.2)
- 📄 Content width slider (500–1100px)
- 🎨 Custom accent colors
- 🟤 Sepia font toggle (light mode)
